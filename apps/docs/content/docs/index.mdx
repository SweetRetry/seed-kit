## 安装

```bash
npm install @seed-kit/ai-sdk-provider
# 或
pnpm add @seed-kit/ai-sdk-provider
```

## 快速开始

```ts
import { seed } from '@seed-kit/ai-sdk-provider';
import { generateText } from 'ai';

const { text } = await generateText({
  model: seed('doubao-seed-1-6-251015'),
  prompt: '你好！',
});

console.log(text);
```

## 配置

通过环境变量设置 API Key：

```bash
ARK_API_KEY=your_api_key_here
```

或在代码中直接传入：

```ts
import { createSeed } from '@seed-kit/ai-sdk-provider';

const seed = createSeed({
  apiKey: 'your_api_key_here',
});
```

## 功能

- **对话生成**（Chat API）：支持流式输出、多模态输入、Function Calling、结构化输出、深度思考
- **响应 API**（Responses API）：新一代对话接口，支持多轮会话管理、联网搜索
- **图像生成**：基于 Seedream 系列模型，支持文生图、图生图、多图融合
- **视频生成**：基于 Seedance 系列模型，支持文生视频、图生视频
